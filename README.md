# Olympus Product Explorer (Prototype)

## Overview
This project is a lightweight prototype of a **Product Explorer application** designed for Olympus product portfolio discovery.

The application enables users to:
- Search products using keywords
- Filter by product family and clinical area
- Explore product details
- View similar/recommended products

The goal is to demonstrate how a simple, user-friendly interface can improve product visibility and accessibility.

---

## Key Features

- 🔍 Keyword-based search (name + description)
- 🎯 Multi-level filtering (Product Family, Clinical Area, Product Type)
- 📊 Interactive product table
- 📄 Product detail view
- ⭐ Recommended similar products
- 🖥️ Clean, enterprise-style UI

---

## Technology Stack

- Python
- Streamlit
- Pandas
- CSV-based dataset (can be replaced with database)
